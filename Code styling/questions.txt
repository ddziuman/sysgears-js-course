Запитання для самоконтролю:

1. Як рекомендується оголошувати змінні, функції та класи? Які існують скорочення при оголошенні властивостей та методів об'єктів?
2. Підготуйте приклад класу з JSDoc коментарями, що знаходиться в модулі, що має конструктор, поля, а також методи зі значеннями, заданими за замовчуванням.
3. Опишіть правила завдання вертикальних відступів для методів класів та об'єктів, а також для коду всередині методів та функцій.
4. Розкажіть про те, як працює деструктуризація у JavaScript, у яких випадках її можна використовувати замість методів Array.prototype?
5. Опишіть правила застосування фігурних дужок, у яких випадках їх можна упускати після if та інших виразів? Як правильно задавати порожні блоки {} для багатоблочних виразів?
6. Коли рекомендується використовувати необов'язкову кому при декларуванні Array та Object літералів?
7. Опишіть особливості використання for..in та for..of виразів, у чому небезпека використання for..in для об'єктів?
